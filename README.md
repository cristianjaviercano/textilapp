# TextileFlow Scheduler Offline

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Deploying to GitHub Pages

This project is configured for automated deployment to GitHub Pages.

### Final Setup Steps

After pushing your code to the `main` branch, you need to configure your GitHub repository to serve the deployed files:

1.  Go to your repository settings on GitHub.
2.  Navigate to the **Pages** section.
3.  Under "Build and deployment", set the **Source** to **GitHub Actions**.

The application will be available at `https://cristianjaviercano.github.io/textilapp`.
